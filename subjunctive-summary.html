英語の仮定法まとめ

仮定法は「事実ではないこと」「まだ実現していないこと」「実現してほしいこと」を表すために、 あえて時制をずらしたり、動詞の原形を使ったりする表現です。

全体像

種類 意味 基本形 例
仮定法過去 現在・未来について「もし今そうなら」と、事実と違うことや可能性の低いことを言う。 If + 主語 + 過去形, 主語 + would/could/might + 原形 If I were rich, I would travel around the world.
仮定法過去完了 過去について「もしあの時そうだったなら」と、過去の事実と違うことを言う。 If + 主語 + had + 過去分詞, 主語 + would/could/might + have + 過去分詞 If I had studied harder, I would have passed the exam.
過去完了 仮定法ではなく、過去のある時点より前に起きたことを表す普通の時制。 主語 + had + 過去分詞 When I arrived, the train had already left.
仮定法現在 要求・提案・決定・命令・必要性などの内容を、that節の中で「そうせよ」という感じで表す。 主語 + 動詞/形容詞 + that + 主語 + (should) + 原形 They demanded that he be present. / It is important that she study.
仮定法未来 未来について「万一」「仮に」という低い可能性・強い仮定を表す。 If + 主語 + should + 原形 / If + 主語 + were to + 原形 If it should rain tomorrow, the game will be canceled.

1. 仮定法過去

現在の反実・可能性の低い未来

If + S + 過去形, S + would / could / might + 動詞の原形

名前は「過去」ですが、意味は主に「今の事実と違うこと」です。 be動詞は、正式には主語に関係なくwereを使うのが基本です。

If I were you, I would apologize.
もし私があなたなら、謝るだろう。

2. 仮定法過去完了

過去の反実

If + S + had + 過去分詞, S + would / could / might + have + 過去分詞

「実際にはそうしなかった・そうならなかった」という過去への後悔、推量、別の可能性を表します。

If she had left earlier, she could have caught the train.
もし彼女がもっと早く出ていたら、その電車に間に合ったかもしれない。

3. 過去完了との違い

普通の過去完了

過去のある時点よりさらに前の出来事を表します。反実の意味はありません。

When I got home, my brother had eaten dinner.

私が帰宅した時、弟は夕食を食べ終えていた。

仮定法過去完了

had + 過去分詞の形を使って、過去の事実と違う仮定を表します。

If I had known the truth, I would have told you.

もし真実を知っていたら、あなたに話していただろう。

4. 仮定法現在

that節の中で「原形」を使う

要求・提案・決定・命令など + that + S + (should) + 動詞の原形

仮定法現在は、特に要求・提案・決定・命令・必要性を表す語の後で使われます。 that節の中では、三単現の-sや過去形にせず、動詞の原形を使います。 イギリス英語ではshould + 原形もよく使われます。

  • 要求: demand, request, require, ask
  • 提案: suggest, propose, recommend
  • 決定: decide, agree, arrange
  • 命令: order, command, insist, urge
  • 形容詞: important, necessary, essential, vital, imperative

The teacher suggested that he study harder.
先生は、彼がもっと一生懸命勉強するよう提案した。

It is necessary that everyone be on time.
全員が時間通りに来ることが必要だ。

命令法だと考える説明: 仮定法現在のthat節は、「事実を述べる節」というより、要求・提案・命令の中身を表す節です。 つまりHe studies.のように「彼は勉強する」と事実を言っているのではなく、 Study.に近い「勉強せよ」という命令法的な内容を、that節の中に入れていると考えると理解しやすくなります。 そのため、主語がheでもstudy、主語がeveryoneでもbeのように原形になります。

5. 仮定法未来

should 万一

If + S + should + 原形, S + will / would / can / could + 原形

未来の可能性が低いことを「万一」と表します。主節には直説法のwillも、仮定法のwouldも来ます。

If you should need help, call me.
万一助けが必要なら、私に電話してください。

were to 仮に

If + S + were to + 原形, S + would / could / might + 原形

かなり起こりにくい未来や、仮に起きたら大きな影響がある未来を表します。

If the sun were to disappear, life would be impossible.
仮に太陽が消えたら、生命は存在できないだろう。

覚え方

  • 今と違う: If + 過去形, would + 原形
  • 昔と違った: If + had + 過去分詞, would have + 過去分詞
  • 要求・提案・命令の中身: that + S + 原形、または should + 原形
  • 未来の万一: If + S + should + 原形 / were to + 原形
